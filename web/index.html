The High-Performance Open-Source Chess Engine Built with Rust | Engineered by Quantum Leaf Automation

ChesseQ is a modern, memory-safe chess engine designed for speed and tactical depth. By leveraging Magic Bitboards and a Functional State architecture, ChesseQ provides a rock-solid foundation for both TUI-based terminal play and WebAssembly-powered browser experiences.
